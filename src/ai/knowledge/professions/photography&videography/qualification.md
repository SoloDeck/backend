# Photography & Videography Qualification Guide

## Purpose

Defines profession-specific interpretation rules for qualifying photography and videography inquiries. This guide supplements the Lead Qualification Framework and does not define scoring.

---

## 1. Domain Scope

### Supported Engagements

- Portrait photography
- Wedding photography
- Event photography
- Product photography
- Food photography
- Corporate photography
- Real estate photography
- Commercial videography
- Social media content production
- Promotional videos

### Out-of-Scope Engagements

- Graphic design
- UI/UX design
- Software development
- Digital marketing strategy
- Copywriting
- Event planning

---

## 2. Intake Field Interpretation

### Project Type

Treat the selected project type as the primary deliverable.

| Project Type | Typical Deliverable |
|--------------|---------------------|
| Portrait | Portrait photo session |
| Wedding | Wedding photography or videography |
| Event | Event coverage |
| Product | Product photography |
| Food | Food photography |
| Corporate | Corporate media |
| Real Estate | Property photography |
| Commercial Video | Promotional video |
| Social Media Content | Short-form content |

If the inquiry conflicts with the selected project type, flag the inconsistency.

---

### Purpose

Treat the selected purpose as the primary creative objective.

| Purpose | Typical Focus |
|---------|---------------|
| Marketing | Promotional content |
| Personal | Personal memories or portraits |
| Corporate | Business communication |
| Documentation | Event or project coverage |

Multiple unrelated purposes may indicate multiple engagements.

---

### Location

Treat the selected location as the expected shooting environment.

| Location | Interpretation |
|----------|----------------|
| Studio | Controlled studio session |
| Client Location | On-site production |
| Outdoor | Outdoor shooting |
| Not Decided | Clarification required |

If the inquiry requests multiple locations, determine whether they belong to a single production or separate engagements.

---

### Required Deliverables

Treat selected deliverables as the strongest indicator of project scope.

| Deliverables | Typical Scope |
|--------------|---------------|
| Edited Photos | Standard photography delivery |
| Raw Photos | Unedited originals |
| Highlight Video | Short edited video |
| Full Video | Complete event or production video |
| Short-form Social Media Clips | Platform-ready content |

Large combinations of deliverables may indicate a more complex production.

---

### Estimated Duration

Treat the estimated duration as the primary indicator of production effort.

| Duration | Interpretation |
|----------|----------------|
| Half Day | Short session |
| One Day | Standard shoot |
| Multiple Days | Extended production |
| Other | Clarification required |

If the stated work appears inconsistent with the selected duration, recommend clarification.

---

## 3. Qualification Rubric

### Strong Qualification Evidence

- Project type identified
- Purpose identified
- Location identified
- Required deliverables identified
- Estimated duration identified

---

### Requires Clarification

- Generic request ("need a photographer")
- Unclear project purpose
- Undecided shooting location
- Undefined deliverables
- Undefined duration
- Conflicting structured fields and inquiry

---

### Scope Consistency

Evaluate whether the structured fields describe a single coherent production.

| Situation | Interpretation |
|-----------|----------------|
| Wedding + Client Location + Full Video | Consistent |
| Product + Studio + Edited Photos | Consistent |
| Wedding + Product + Real Estate | Possible multiple engagements |
| Half-day selected but inquiry requests full wedding coverage with video | Inconsistent |

---

### Feasibility Signals

Identify, but do not reject:

- Extensive deliverables within limited duration
- Unrealistic production schedule
- Multiple unrelated productions
- Insufficient information for estimating production effort

---

## 4. Mixed-Domain Guidance

Separate photography and videography work from requests belonging to other professions.

| Request | Qualification |
|---------|---------------|
| Photograph product catalog | Photography & Videography |
| Edit a company logo | Out of Scope |
| Run social media advertising | Out of Scope |
| Build an e-commerce website | Out of Scope |

Qualify only the photography and videography portion and flag remaining requests for manual assessment.
