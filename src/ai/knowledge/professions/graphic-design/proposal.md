# Graphic Design Proposal Guide

## 1. Purpose

Defines profession-specific guidance for generating graphic design proposals from qualified leads. This guide supplements the proposal generation framework and does not define proposal formatting.

---

## 2. Supported Project Types

Generate proposals for engagements including:

- Logo design
- Brand identity
- Social media graphics
- Posters
- Brochures
- Packaging
- Presentation design
- Illustration
- Marketing materials

---

## 3. Proposal Structure

A graphic design proposal should generally include:

1. Project summary
2. Design scope
3. Deliverables
4. Timeline
5. Pricing
6. Assumptions
7. Exclusions

---

## 4. Scope Interpretation

### Converting Client Requirements into Scope

Convert qualified intake information into design scope.

| Intake Information | Proposal Scope |
|-------------------|----------------|
| Design category | Primary design service |
| Business goal | Design objective |
| Target users | Intended audience |
| Brand assets available | Existing design resources |
| Expected deliverables | Design outputs |

---

### Handling Missing Information

If important information is unavailable:

- State it as a clarification item.
- Do not invent branding requirements or design assets.
- Limit the proposal to confirmed project scope.

---

### Handling Assumptions

Distinguish between:

- Confirmed requirements
- Reasonable design assumptions
- Optional design recommendations

Do not present assumptions as confirmed project requirements.

---

## 5. Deliverable Guidance

### Typical Deliverables

Depending on project scope, deliverables may include:

- Logo concepts
- Final logo package
- Brand identity assets
- Social media graphics
- Print-ready artwork
- Packaging artwork
- Presentation slides
- Illustrations
- Source files
- Brand guidelines

---

### Deliverable Wording

Describe deliverables as completed design assets.

Prefer:

- Logo package
- Brand identity system
- Print-ready brochure
- Social media asset set

Avoid:

- Design logo
- Make poster
- Draw illustrations

---

### Optional Deliverables

Only include when supported by the inquiry:

- Source files
- Brand guideline
- Print-ready files
- Social media adaptations
- Additional revision rounds
- Multiple file formats

---

## 6. Timeline Guidance

### Common Project Phases

Typical phases include:

- Requirements confirmation
- Concept development
- Initial design
- Client revisions
- Final artwork preparation
- File delivery

---

### Timeline Estimation Guidance

Estimate timelines according to:

- Design complexity
- Number of deliverables
- Revision rounds
- Brand asset availability

Avoid precise schedules when project scope is incomplete.

---

### Dependency Considerations

Identify dependencies including:

- Client feedback
- Brand assets
- Content provided by client
- Revision approvals
- Printing specifications (if applicable)

---

## 7. Pricing Guidance

### Pricing Factors

Consider factors such as:

- Design category
- Number of deliverables
- Original illustration work
- Branding complexity
- Revision requirements
- Licensing requirements

---

### What Affects Estimates

Estimated pricing may increase due to:

- Complete brand identity systems
- Large illustration projects
- Multiple deliverable formats
- Extensive revisions
- Commercial licensing requirements

---

### When Not to Estimate

Avoid estimating when:

- Design scope is unclear.
- Required deliverables are undefined.
- Brand assets or project requirements remain uncertain.

Recommend clarification instead.

---

## 8. Common Assumptions

### Safe Assumptions

Unless stated otherwise, proposals may reasonably assume:

- Digital delivery
- Standard file formats
- Industry-standard design software
- Typical commercial usage

---

### Assumptions Requiring Confirmation

Do not assume:

- Number of concepts
- Revision rounds
- Print production
- Commercial usage rights
- Source file delivery
- Brand guidelines
- Existing visual identity

These should be confirmed with the client.

---

## 9. Out-of-Scope Requests

Do not include services outside graphic design unless explicitly requested and supported.

Examples include:

- UI/UX design
- Software development
- Marketing strategy
- Copywriting
- Photography
- Video production

When mixed-domain requests are identified, generate the proposal only for the graphic design portion and identify remaining requests separately.
