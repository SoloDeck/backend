# Graphic Design Qualification Guide

## Purpose

Defines profession-specific interpretation rules for qualifying graphic design inquiries. This guide supplements the Lead Qualification Framework and does not define scoring.

---

## 1. Domain Scope

### Supported Engagements

- Logo design
- Brand identity design
- Social media graphics
- Posters
- Brochures
- Packaging design
- Presentation design
- Illustration
- Marketing collateral
- Print design

### Out-of-Scope Engagements

- UI/UX design
- Software development
- Digital marketing
- Copywriting
- Photography and videography
- Architecture and industrial design

---

## 2. Intake Field Interpretation

### Design Category

Treat the selected design category as the primary project deliverable. If the inquiry conflicts with the selected category, flag the inconsistency.

| Design Category | Typical Deliverable |
|-----------------|---------------------|
| Logo | Logo package |
| Brand Identity | Complete branding system |
| Social Media Graphics | Social media asset set |
| Poster | Print or digital poster |
| Brochure | Marketing brochure |
| Packaging | Packaging artwork |
| Presentation | Presentation deck |
| Illustration | Custom illustrations |

---

### Business Goal

Treat the business goal as the primary indicator of design intent.

Interpret goals as communication or branding objectives rather than visual styles.

| Business Goal | Typical Design Focus |
|---------------|----------------------|
| Build brand recognition | Brand identity |
| Promote a campaign | Marketing materials |
| Increase product appeal | Packaging design |
| Improve professional image | Corporate branding |

Multiple unrelated business goals may indicate multiple design engagements.

---

### Target Users

Use the target users to determine the intended audience for the design.

Examples:

- Consumers
- Business clients
- Students
- Internal employees
- Event attendees

Missing target users reduce confidence in design direction.

---

### Brand Assets Available

Use available brand assets to estimate design starting point.

| Assets | Interpretation |
|--------|----------------|
| Yes | Existing branding should be reused |
| Partial | Some branding exists; clarification may be required |
| None | New branding or visual direction may be required |

If branding is required but no assets exist, recommend clarification.

---

### Expected Deliverables

Treat selected deliverables as the strongest indicator of project scope.

| Deliverables | Typical Scope |
|--------------|---------------|
| Source Files | Editable project files |
| Print-ready Files | Production-ready assets |
| Social Media Versions | Platform-specific exports |
| Multiple Revisions | Iterative design process |
| Brand Guideline | Brand usage documentation |

Large combinations of unrelated deliverables may indicate multiple design phases.

---

## 3. Qualification Rubric

### Strong Qualification Evidence

- Design category identified
- Business goal clearly stated
- Target users identified
- Brand assets identified
- Expected deliverables identified

---

### Requires Clarification

- Generic request ("need a design")
- Unclear business goal
- Missing target users
- Unknown brand assets
- Undefined deliverables
- Conflicting structured fields and inquiry

---

### Scope Consistency

Evaluate whether the structured fields describe a single coherent design engagement.

| Situation | Interpretation |
|-----------|----------------|
| Logo + Brand Guideline + Source Files | Consistent |
| Packaging + Print-ready Files | Consistent |
| Logo + Packaging + Illustration + Presentation | Possible multiple engagements |
| Existing brand assets selected but inquiry requests complete rebranding | Inconsistent |

---

### Feasibility Signals

Identify, but do not reject:

- Large branding scope with minimal budget
- Unrealistic delivery timeline
- Multiple unrelated design projects
- Insufficient information for estimating design effort

---

## 4. Mixed-Domain Guidance

Separate graphic design work from requests belonging to other professions.

| Request | Qualification |
|---------|---------------|
| Design a company logo | Graphic Design |
| Build a company website | Out of Scope |
| Write advertising copy | Out of Scope |
| Run Facebook Ads | Out of Scope |

Qualify only the graphic design portion and flag remaining requests for manual assessment.
